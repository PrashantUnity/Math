Introduction to Algebra#

When you first start mathematics in primary school, you deal with numbers like 5, 12, or 100. But what happens when we don’t know the exact number yet? That is where Algebra comes in!

Algebra introduces letters (variables) to represent unknown quantities.

1. Constants vs. Variables#

  • Constant: A symbol that has a fixed numerical value.
    • Examples: $5$, $-3$, $\frac{1}{2}$, $\pi \approx 3.14$
  • Variable: A letter or symbol used to represent a value that can change or is unknown.
    • Examples: $x$, $y$, $a$, $b$, $n$

Real-World Example:#

Imagine you buy $x$ notebooks for $3 each. The total cost is: $$\text{Total Cost} = 3 \times x = 3x$$

If $x = 4$ notebooks, total cost = $12. If $x = 10$ notebooks, total cost = $30.

2. Algebraic Expressions & Terms#

An algebraic expression is formed by combining constants and variables using operations like addition, subtraction, multiplication, and division.

3. Solving Simple Linear Equations#

An equation is a mathematical statement showing that two expressions are equal ($=$).

Problem: Solve for $x$#

$$2x + 5 = 15$$

Step-by-Step Solution:#

  1. Subtract 5 from both sides: $$2x + 5 - 5 = 15 - 5$$ $$2x = 10$$
  2. Divide by 2 on both sides: $$\frac{2x}{2} = \frac{10}{2}$$ $$x = 5$$

🎉 Answer: $x = 5$. Check your answer: $2(5) + 5 = 10 + 5 = 15$. Correct!

Practice Questions for Class 6 Students#

  1. Solve for $y$: $3y - 4 = 11$
  2. Write an algebraic expression for: “5 added to twice a number $n$”
  3. If $a = 3$ and $b = 4$, find the value of $2a + 3b$.
